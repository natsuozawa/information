Palantir Technologiesでインターンシップを６月より始めます

2023年6月3日

Palantir Technologiesの東京オフィスでForward Deployed Software Engineer (FDSE)として３ヶ月間インターンします。ソフトウェアエンジニアとプロダクトマネージャー両方として動き、日本の企業が抱える重要なビジネスと技術の課題の解決に貢献する仕事です。

Palantirについて

PalantirはアメリカでPayPal創業者のPeter Thielが共同創業した会社で、政府や大企業を顧客としてあらゆる業界のデジタルトランスフォーメーションに取り組んでいます。日本オフィスはまだ小さくスタートアップみたいな環境だと聞いていますが、今後日本市場の開拓に貢献できればと思っています。

DXに取り組んでいる企業はたくさんありますが、Palantirの技術的知見とビジネスモデルは特殊だと思います。最初の顧客はアメリカの情報機関と軍であり、機密性の高い大量のデータを解析し、瞬時に様々な意思決定を行えるように汎用的なソフトウェアを開発しました。

最近では、世界中の製造業、ヘルスケア、官公庁、金融、物流などの業界でシステムがデプロイされています。Palantirは業務プロセスをスマート化することで、Business as Codeと呼ばれる、ビジネスがコードのよう（正確・アジャイル・意思決定が速い・スケーラブル）に振る舞うことを可能にします。

Palantirは顧客企業にFDSEと呼ばれるソフトウェアエンジニアを送り込み、顧客の業務プロセス変革とエンジニアリングの課題を見つけ、米国・英国で開発したプロダクトと顧客企業のシステムを統合することでDXを強力に支援します。

このビジネスモデルを通してレベルの高いソフトウェアの力を日本企業に提供できると考えています。
